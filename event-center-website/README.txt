# Concert Center Website

## Overview
This project is a multi-page website for a live music concert venue, built using HTML and CSS. It features a homepage, events listing, food & drinks menu, location info, and a contact form. The design is fully responsive and visually themed for concerts and music lovers.

## File Structure
- `index.html` — Home page with hero image, welcome, about, and featured events
- `events.html` — Upcoming events, each with image and description
- `menu.html` — Menu organized by categories, with images and prices
- `location.html` — Address, hours, and map/landmarks
- `contact.html` — Contact form with validation
- `css/style.css` — All site styling (responsive, interactive)
- `images/` — Folder for hero, event, menu, and map images

## Features
- Responsive navbar on every page
- Hero image & featured events on home page
- Events page using semantic HTML
- Menu page with a styled, interactive table
- Location page with address and map placeholder
- Contact form with required fields and clean styling
- Fully responsive layout for all devices

## How to Use
1. Open `index.html` in your browser to start.
2. Navigate with the menu to explore all pages.
3. Replace images in the `images/` folder to personalize the theme.
4. To embed a real Google Map, replace the map image on `location.html` with an embed code.

## Testing
- Check the site on different devices to see responsive design in action.
- Submit the contact form with and without filling fields to see validation.
- Hover over menu/table rows and navigation links for interactive effects.
